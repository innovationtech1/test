Desarrollo Web

Desarrollamos tu paginas web personalizada.

Que necesito para comenzar?
Solo necesitamos un hosting un dominio para poder empezar a crear tu pagina web.

Buscamos un dominio apropiado (un dominio es la direccion para poder buscar tu pagina en internet "www.ejemplo tudominio.com") una vez que tengamos el dominio necesitamos configurarlo en un servidor web tambien llamado "Hosting" es alli donde estara hospedada la informacion de tu pagina.

Como creamos tu pagina?
Nosotros trabajamos con tegnologias web como HTML5, CSS3 Y JAVASCRIP y utilizamos estas tecnologias para poder escribir el codigo necesario para darle vida a tu pagina despues lo configuramos en un hosting para que pueda ser visitado


Que puedo poner en mi sitio web?
Podemos publicar cualquier tipo de contenido multimedia como texto, fotos ,videos, enlazar redes sociales y correo electronico. (los costos varian de acuerdo al contenido).
